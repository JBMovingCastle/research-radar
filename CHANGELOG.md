# Changelog

## 0.1.1 — 2026-07-21

- Kept the default daily selection at 5 while allowing users to configure 1–20 items.
- Added editable report requirements, item-field order, abstract length, opening note, and section headings in `report` configuration.
- Removed presentation materials from the public default branch.

## 0.1.0 — 2026-07-21

- Added no-key first-run configuration and interactive initialization.
- Added config validation, repository-safe output paths, and environment-only secrets.
- Added Crossref, arXiv, Semantic Scholar Search, Semantic Scholar Recommendations, OpenAlex, RSS/Atom, and official-page adapters.
- Added author/ORCID, seed-paper, target-journal/ISSN, `prefer`/`only`, scoring, rotation, and output-directory configuration.
- Added normalized candidates, provider isolation, partial-coverage reporting, deduplication, scoring, daily idempotency, and recommendation ledger.
- Added deterministic Markdown briefs plus local, stdout, and optional Feishu delivery.
- Added two Codex Skills, public documentation, offline tests, CI workflow, sanitized example, and six-slide sharing deck.
